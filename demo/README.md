# KIWI Static Demo

This folder contains a static HTML demo for:

Knowledge Intake Workbench Intelligence - KIWI

## Files

- `index.html` - demo entry page
- `styles.css` - demo styling
- `app.js` - simulated interactions
- `assets/kiwi-bird.png` - KIWI logo image

## Hostinger Upload

Upload the contents of this `demo` folder to your Hostinger site.

For the main domain:

```text
public_html/
  index.html
  styles.css
  app.js
  assets/
```

For a subfolder such as `yourdomain.com/kiwi-demo/`:

```text
public_html/
  kiwi-demo/
    index.html
    styles.css
    app.js
    assets/
```

No backend, database, Node.js server, or Python service is required. The demo uses static sample data only.
